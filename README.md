# BiliTune
### 这是什么？
```
模仿Youtube music的思路、Spotify的前端制作的哔哩哔哩音乐播放器
```
### 有什么特点？
```
- 支持Windows、Android双端（或许还会支持更多？）
- 把B站账号的收藏夹当作歌单，在app内可以创建新的收藏夹，添加视频进入收藏
- 从视频字幕获取歌词
- 好看的ui/ux，轻松易懂的使用方式
```
