</div>

### 这是什么？
```
模仿Youtube music的思路、Spotify的前端制作的哔哩哔哩音乐播放器
```
### 有什么特点？
```
- 支持Windows、Android双端（或许还会支持更多？）
- 把B站账号的收藏夹当作歌单，在app内可以创建新的收藏夹，添加视频进入收藏
- 从视频字幕或网易yun等平台获取歌词
- flutter制作，多端支持优异，内存占用小
- 好看的ui/ux，轻松易懂的使用方式
- 尊重Bilibili bot协议，严格限制api调用频率，避免为上游增加压力以及触发风控
```
