核心功能

为什么选择 BiliTune

专为 B站用户设计，融合 Spotify 体验与哔哩哔哩内容生态

📂
收藏夹即歌单
将你的 B站收藏夹直接当作歌单使用，也可以在应用内创建新收藏夹、添加视频。
📝
字幕提取歌词
从视频字幕自动获取歌词并逐行同步高亮，带你重新发现熟悉的旋律。
💻
Windows + Android
Flutter 跨平台开发，Windows 桌面与 Android 手机体验高度一致，随时切换。
🎨
沉浸式深色 UI
专辑封面渐变提色，动态背景随当前歌曲自动变化，界面退居幕后让内容成为主角。
🎬
视频 + 音频双源
同时支持 B站视频区与音频区内容，用 BV 号或关键词直接搜索，一个 App 全搞定。
⚡
流畅微交互
200ms ease 过渡动画，悬停浮现播放按钮，进度条智能扩展，每一处细节都经过打磨。
